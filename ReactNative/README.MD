## Installation

- clone this repo
- enter the sandbox credentials in App.js
  - senderId
  - clientId
  - clientSecret 
- from the terminal
  - `cd ReactNative`
  - `npm install`
  - `cd ios`
  - `pod install`
  - `cd ..`
  - `npx react-native start`
- from a second terminal 
  - `npm run <platform> (android | ios)`

## Verified Platforms
- M1, M2, Intel on OSX Ventura 13.3.1
- Xcode 14.2, 14.3
- Android Studio Electric Eel 2022.1.1, Flamingo 2022.2.1
