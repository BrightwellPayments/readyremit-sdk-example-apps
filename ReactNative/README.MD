## Installation

- clone this repo
- enter the sandbox credentials in App.js
  - senderId
  - clientId
  - clientSecret 
- from the terminal
  - `cd ReactNative`
  - `npm install`
  - `cd ios`
  - `pod install`
  - `cd ..`
  - `npx react-native start`
- from a second terminal 
  - `npm run <platform> (android | ios)`
